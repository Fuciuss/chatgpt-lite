About the package

The Community presentation package has been designed to assist you to teach these community members about Australian workplace laws. Anyone can use these resources - you donâ€™t need to be an expert.

The package includes:

  • Working in Australia presentation slides:
    • a comprehensive version, which includes detailed speaking notes and optional slides that give an overview of the FWO website. This presentation gives you the ability to tailor a presentation for various levels of audience
    • a shorter version, also with detailed speaking notes to be used with our Working in Australia video. This presentation does not contain the FWO website overview and is specifically tailored for students and newly arrived migrant workers
  • Working in Australia video
  • A guide to starting a new job
  • Fair Work Information Statement
  • Casual Employment Information Statement

If you need further guidance on using this package please contact us .
